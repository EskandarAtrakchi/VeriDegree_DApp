# 🚀 VeriDegree DApp - START HERE

**Welcome!** You now have a complete educational certificate NFT platform.

## ⚡ 30-Second Summary

This is a web3 app that:
- Issues certificates as NFTs on Ethereum
- Stores files on Pinata IPFS
- Verifies certificates on blockchain
- Works with MetaMask wallet
- Runs on your PC locally
- Everything is pre-configured and ready to go

## 📋 What You Need to Do

### 1️⃣ Download the Project (1 min)
- [ ] Download ZIP from v0 (top right button)
- [ ] Extract to your computer
- [ ] Remember the folder location

### 2️⃣ Install Dependencies (2 min)
```bash
cd VeriDegreeDAppmain
pnpm install
```
(or `npm install` if pnpm not available)

### 3️⃣ Create Environment File (1 min)
```bash
cp .env.local.example .env.local
```
**That's it!** Credentials are already filled in.

### 4️⃣ Setup MetaMask (5 min)
1. Install MetaMask: https://metamask.io/
2. Switch to Sepolia testnet
3. Get free testnet ETH: https://www.sepoliafaucet.com/

### 5️⃣ Run the App (1 min)
```bash
pnpm dev
```
Visit: **http://localhost:3000**

## ✅ You're Done!

Now you can:
- Connect wallet
- Issue certificates
- Verify certificates
- View certificate details

**Total setup time: ~15 minutes**
